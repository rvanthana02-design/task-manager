# Task Manager Web Application 📝

A sleek, lightweight, and functional Task Manager built using pure HTML, CSS, and JavaScript. This project is designed to help users organize their daily tasks with an intuitive interface and smooth user experience.

## 🌟 Features

* **Create Tasks:** Easily add new tasks with a title to your list.
* **Status Toggle:** Click on any task to mark it as completed (strikethrough effect).
* **Delete Functionality:** Remove tasks instantly using the dedicated delete button.
* **Responsive Design:** Fully optimized for all screen sizes, including mobile, tablet, and desktop.
* **Clean UI:** Modern aesthetic with a focus on readability and simplicity.

## 🛠️ Technologies Used

* **HTML5:** For semantic web markup.
* **CSS3:** For custom styling, flexbox layouts, and UI transitions.
* **JavaScript (ES6):** For DOM manipulation and application logic.

## 🚀 Getting Started

To run this project locally, follow these simple steps:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/your-username/task-manager.git](https://github.com/your-username/task-manager.git)
    ```
2.  **Navigate to Folder:**
    ```bash
    cd task-manager
    ```
3.  **Launch the App:**
    Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, etc.).
